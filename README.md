# Travel Buddy

Travel Buddy is a trip-planning web application that helps users organize travel plans, manage itineraries, and keep trip information in one place.

## Features

### Trip Dashboard
- View all planned trips in a clean dashboard
- Search trips by name
- Filter trips by status (Upcoming, Ongoing, Past)
- Filter trips by category
- Sort trips by creation date

### Trip Creation
- Create new trips through an interactive form
- Add destination, dates, category, and description
- Select a custom cover image for each trip

### Trip Cards
- Visual trip cards with cover images
- Progress tracking for trip completion
- Category labels and trip status indicators
- Quick access to trip details

### Trip Details
- Dedicated trip detail page
- View trip information and progress
- Edit or duplicate existing trips
- Delete trips when no longer needed

### Timeline Planning
- Day-by-day itinerary view
- Organize activities by date
- Track completed activities
- Add activities directly to the trip timeline

---

- Next.js 15
- React
- TypeScript
- Material UI (MUI)
